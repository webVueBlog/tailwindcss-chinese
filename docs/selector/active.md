伪选择器在:active链接被激活（被点击或以其他方式激活）时更改链接的外观。它通常只出现一瞬间，并提供元素确实被点击的视觉反馈。它最常用于锚链接 ( `<a href="#"></a>`)。

例如，下面的 CSS 将使锚链接在活动状态下向下一个像素（给人一种被推入 3D 空间的印象）：

顺序执行：

```js
a:link { /* Essentially means a[href], or that the link actually goes somewhere */
  color: blue;
}
a:visited {
  color: purple;
}
a:hover {
  color: green;
}
a:active {
  color: red;
}
```
